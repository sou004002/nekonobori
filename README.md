# ねこのぼり
unityを用いて作成したジャンプアクションゲームです。

# Features
awsdキー、または矢印キーで移動、スペース長押し→離してジャンプです。

# Usage
[unityroomでプレイ](https://unityroom.com/games/nekonobori)<br>
